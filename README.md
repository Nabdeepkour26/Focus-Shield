# Focus Shield 🛡️ - The Ultimate Focus & Productivity Suite

![Focus Shield Banner](frontend/images/guardian.png)

> **"Transform your study sessions into legendary focus battles. Built by students, for students."**

Focus Shield is a **dual-powered productivity system** that combines cutting-edge web technologies with gamified discipline mechanics. Choose between our **premium HTML/CSS/JS version** or our **advanced React version** - both designed to help you conquer distractions and achieve deep focus.

---

## 🎯 **Choose Your Weapon**

### **⚡ Frontend Version** (`/frontend`)
- **Pure HTML/CSS/JavaScript** - Lightning fast, no dependencies
- **Cyber-Midnight aesthetics** with glassmorphism effects
- **Slide-form navigation** - Immersive presentation-style interface
- **Perfect for**: Quick deployment, GitHub Pages, static hosting

### **🚀 React Version** (`/react-app`)
- **Modern React 18 + Vite** - Advanced component architecture
- **Real-time distraction detection** with browser APIs
- **Dynamic score engine** with persistent storage
- **Multiple motivational personas** (Strict, Friendly, Military)
- **Perfect for**: Full-featured apps, Vercel/Netlify deployment

---

## 🌟 **Core Features**

### **🛡️ Focus Guardian System**
- **AI-like mascot** that provides real-time status updates
- **Neural breathing timer** (6 breaths/minute) for calm focus
- **Session extension penalties** (+2min) for every tab switch

### **🎮 Gamified Discipline Engine**
- **XP-based scoring system** (0-100 points)
- **Rank progression**: Novice → Zen Monk → Focus Master
- **Achievement unlocks** and milestone rewards
- **Red alert animations** for distraction penalties

### **⚡ Advanced Timer Mechanics**
- **Pomodoro presets**: 25min work + 5min break cycles
- **Custom durations**: Up to 5 hours for deep work sessions
- **Progress visualization** with animated progress bars
- **Session statistics** and completion tracking

### **🎨 Premium UI/UX**
- **Glassmorphism 2.0** with backdrop filters
- **Smooth animations** powered by CSS transforms
- **Responsive design** for all devices
- **Dark theme optimized** for late-night study sessions

## 🚀 **Quick Start**

### **Option 1: Frontend Version (Recommended for Beginners)**
```bash
# Navigate to frontend directory
cd frontend

# Serve locally (Python 3)
python -m http.server 3000

# Or use any static server
# Open http://localhost:3000
```

### **Option 2: React Version (Advanced Features)**
```bash
# Navigate to React app
cd react-app

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
